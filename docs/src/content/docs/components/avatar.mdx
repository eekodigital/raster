## Initials fallback

When no image is provided (or the image fails to load), the fallback renders instead.

<Example
  code={`import { Avatar } from '@eekodigital/raster';\n\n<Avatar.Root>\n  <Avatar.Fallback>JD</Avatar.Fallback>\n</Avatar.Root>`}
  center
>
  <AvatarInitialsDemo client:load />
</Example>

## With image

Provide an `Avatar.Image`. Radix handles the load/error state and shows the `Fallback` if the image fails.

<Example
  code={`<Avatar.Root>\n  <Avatar.Image src="/avatar.jpg" alt="Jane Doe" />\n  <Avatar.Fallback>JD</Avatar.Fallback>\n</Avatar.Root>`}
  center
>
  <AvatarImageDemo client:load />
</Example>

## Sizes

Three sizes: `sm` (2rem), `md` (2.5rem, default), `lg` (3.5rem).

<Example
  code={`<Avatar.Root size="sm"><Avatar.Fallback>SM</Avatar.Fallback></Avatar.Root>\n<Avatar.Root size="md"><Avatar.Fallback>MD</Avatar.Fallback></Avatar.Root>\n<Avatar.Root size="lg"><Avatar.Fallback>LG</Avatar.Fallback></Avatar.Root>`}
  center
>
  <AvatarSizesDemo client:load />
</Example>

## Square

Use `square` for organisation logos and non-person avatars.

<Example
  code={`<Avatar.Root square size="md"><Avatar.Fallback>OG</Avatar.Fallback></Avatar.Root>`}
  center
>
  <AvatarSquareDemo client:load />
</Example>
