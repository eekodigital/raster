## Heading scale

Six heading levels. Levels 1-2 use the serif family (`Newsreader`) for editorial weight; levels 3-6 use the sans-serif family.

<Example
  code={`import { Heading } from '@eekodigital/raster';\n\n<Heading level="1">Heading 1</Heading>\n<Heading level="2">Heading 2</Heading>\n<Heading level="3">Heading 3</Heading>`}
>
  <HeadingScaleDemo client:load />
</Example>

## Decoupling tag from visual level

Use the `as` prop to render a different HTML tag without changing the visual style. Useful when the document outline and visual hierarchy diverge.

<Example
  code={`{/* Visually looks like an h3, but contributes an h2 to the document outline */}\n<Heading level="3" as="h2">Section title</Heading>`}
>
  <HeadingAsDemo client:load />
</Example>
