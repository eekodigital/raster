### Justify

Spread items across the main axis with `justify="between"`.

<Example
  code={`<Flex justify="between">\n  <Button variant="secondary">Cancel</Button>\n  <Button>Save</Button>\n</Flex>`}
>
  <FlexJustifyDemo client:load />
</Example>

### No wrap

Items don't wrap by default. Set `wrap` to enable wrapping.

<Example code={`<Flex gap="3">\n  <Tag>One</Tag>\n  <Tag>Two</Tag>\n  <Tag>Three</Tag>\n</Flex>`}>
  <FlexNoWrapDemo client:load />
</Example>

## Column (vertical)

### Basic

<Example
  code={`<Flex direction="column" gap="4">\n  <div>Item one</div>\n  <div>Item two</div>\n  <div>Item three</div>\n</Flex>`}
>
  <FlexColumnDemo client:load />
</Example>

### Gap sizes

Gaps map directly to spacing tokens (`--spacing-*`).

<Example
  code={`<Flex direction="column" gap="8">\n  <div>Wider gap</div>\n  <div>Between items</div>\n</Flex>`}
>
  <FlexColumnGapDemo client:load />
</Example>

### Alignment

Control cross-axis alignment with the `align` prop.

<Example
  code={`<Flex direction="column" align="center">\n  <Button>Short</Button>\n  <Button>A longer button</Button>\n</Flex>`}
>
  <FlexColumnAlignDemo client:load />
</Example>

### Responsive

Props accept responsive objects with named breakpoints (`sm`, `md`, `lg`, `xl`) or custom pixel values.

```tsx
<Flex direction={{ base: "column", "600px": "row" }} gap={{ base: "2", md: "4" }}>
  <div>First</div>
  <div>Second</div>
</Flex>
```

### Semantic element

Use the `as` prop to change the rendered element.

```tsx
<Flex as="nav" gap="4">
  <a href="/">Home</a>
  <a href="/about">About</a>
</Flex>
```
