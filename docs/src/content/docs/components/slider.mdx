## Basic

<Example
  code={`import { Slider } from '@eekodigital/raster';\n\nconst [value, setValue] = useState([50]);\n\n<Slider.Root value={value} onValueChange={setValue} aria-label="Volume">\n  <Slider.Track />\n  <Slider.Thumb aria-label="Volume" />\n</Slider.Root>`}
>
  <SliderBasicDemo client:load />
</Example>

## Range

Use two thumbs for selecting a range between a minimum and maximum.

<Example
  code={`const [value, setValue] = useState([20, 80]);\n\n<Slider.Root value={value} onValueChange={setValue} aria-label="Price range">\n  <Slider.Track />\n  <Slider.Thumb aria-label="Minimum price" />\n  <Slider.Thumb aria-label="Maximum price" />\n</Slider.Root>`}
>
  <SliderRangeDemo client:load />
</Example>

## Step

The `step` prop snaps the thumb to discrete intervals.

<Example
  code={`<Slider.Root value={value} onValueChange={setValue} min={0} max={100} step={10} aria-label="Steps">\n  <Slider.Track />\n  <Slider.Thumb aria-label="Steps" />\n</Slider.Root>`}
>
  <SliderStepDemo client:load />
</Example>

## Disabled

<Example
  code={`<Slider.Root value={[40]} disabled aria-label="Volume">\n  <Slider.Track />\n  <Slider.Thumb aria-label="Volume" />\n</Slider.Root>`}
>
  <SliderDisabledDemo client:load />
</Example>
