## Basic

Each datum has a `label`, `value`, and `color`. Children render in the centre.

<Example
  code={`import { DonutChart } from '@eekodigital/raster';\n\n<DonutChart\n  data={[\n    { label: 'Pass', value: 42, color: 'var(--color-success)' },\n    { label: 'Fail', value: 8, color: 'var(--color-danger)' },\n    { label: 'N/A', value: 12, color: 'var(--color-inactive)' },\n    { label: 'To do', value: 24, color: 'var(--color-warning)' },\n  ]}\n  aria-label="Conformance summary"\n>\n  <strong>86</strong>\n  <span>criteria</span>\n</DonutChart>`}
>
  <DonutChartBasicDemo client:load />
</Example>

## Small / progress indicator

Use smaller `size` and `thickness` for inline progress indicators.

<Example
  code={`<DonutChart\n  data={[\n    { label: 'Complete', value: 75, color: 'var(--color-success)' },\n    { label: 'Remaining', value: 25, color: 'var(--color-border)' },\n  ]}\n  size={80}\n  thickness={10}\n  aria-label="75% complete"\n>\n  <strong>75%</strong>\n</DonutChart>`}
>
  <DonutChartSmallDemo client:load />

## Accessibility

- SVG has `role="img"` with `aria-label`
- Each segment is focusable with label, value, and percentage announced
- Arrow keys navigate between segments (wraps around)
- A hidden `<table>` provides the data with percentages for screen readers
- Zero-value segments are skipped (not rendered or focusable)
