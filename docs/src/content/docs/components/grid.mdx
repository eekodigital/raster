---
title: Grid
description: CSS Grid layout with flexible column widths. Each Grid.Col child specifies its width using fr units, pixel values, auto, or full. No artificial column system — values pass through to grid-template-columns directly.
---

## Proportional columns

Use `fr` units for proportional sizing. `2fr` takes twice the space of `1fr`.

<Example
  code={`import { Grid } from '@eekodigital/raster';\n\n<Grid gap="4">\n  <Grid.Col width="2fr">Main content</Grid.Col>\n  <Grid.Col width="1fr">Sidebar</Grid.Col>\n</Grid>`}
>
  <GridProportionalDemo client:load />
</Example>

## Equal columns

Give every column `1fr` for equal widths.

<Example
  code={`<Grid gap="4">\n  <Grid.Col width="1fr">Card</Grid.Col>\n  <Grid.Col width="1fr">Card</Grid.Col>\n  <Grid.Col width="1fr">Card</Grid.Col>\n</Grid>`}
>
  <GridEqualDemo client:load />
</Example>

## Fixed + flexible

Mix fixed pixel widths with `auto` to fill remaining space.

<Example
  code={`<Grid gap="4">\n  <Grid.Col width="200px">Fixed sidebar</Grid.Col>\n  <Grid.Col width="auto">Fills remaining</Grid.Col>\n</Grid>`}
>
  <GridFixedDemo client:load />
</Example>

## Responsive

Column widths accept responsive objects with named breakpoints or custom pixel values.

<Example
  code={`<Grid gap="4">\n  <Grid.Col width={{ base: 'full', md: '2fr' }}>Main</Grid.Col>\n  <Grid.Col width={{ base: 'full', md: '1fr' }}>Sidebar</Grid.Col>\n</Grid>`}
>
  <GridResponsiveDemo client:load />
</Example>
