---
title: Textarea
description: "Two levels of the same control. Textarea is the bare styled element — use it when you need custom layout or to attach accessories. TextareaField adds a label, optional hint, optional error, and optional character counter."
---

## TextareaField

### Default

<Example
  code={`import { TextareaField } from '@eekodigital/raster';\n\n<TextareaField label="Notes" name="notes" rows={4} />`}
>
  <TextareaField client:load label="Notes" name="notes" rows={4} />
</Example>

### With hint

<Example
  code={`<TextareaField label="Notes" name="notes" hint="Add any observations about this criterion." rows={4} />`}
>
  <TextareaField
    client:load
    label="Notes"
    name="notes"
    hint="Add any observations about this criterion."
    rows={4}
  />
</Example>

### With error

<Example
  code={`<TextareaField label="Notes" name="notes" error="Notes cannot be empty." rows={4} />`}
>
  <TextareaField client:load label="Notes" name="notes" error="Notes cannot be empty." rows={4} />
</Example>

### Character count

Pass `maxLength` to display a live character counter. It switches to error styling when the limit is exceeded.

<Example
  code={`<TextareaField label="Description" maxLength={200} hint="Describe the issue in detail" />`}
>
  <TextareaCharCountDemo client:load />
</Example>

<Example
  code={`<TextareaField label="Short note" maxLength={20} defaultValue="This text is too long and will overflow the limit" />`}
>
  <TextareaCharCountOverDemo client:load />
</Example>
