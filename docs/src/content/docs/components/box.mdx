title: Box
description: A polymorphic element with token-constrained padding. Use Box when you need spacing without creating a custom CSS class. All padding values generate hashed vanilla-extract classes — no inline styles.
---

## Padding

The `padding` prop applies equal padding on all sides using the spacing token scale.

<Example
  code={`import { Box } from '@eekodigital/raster';\n\n<Box padding="4">\n  Content with padding "4" (1rem)\n</Box>`}
>
  <BoxPaddingDemo client:load />
</Example>

### Directional padding

Use `paddingX` and `paddingY` for horizontal and vertical padding independently.

<Example
  code={`<Box paddingX="6" paddingY="3">\n  Different horizontal and vertical padding\n</Box>`}
>
  <BoxPaddingXYDemo client:load />
</Example>

## Semantic element

Use the `as` prop to render as any HTML element.

<Example code={`<Box as="section" padding="4">\n  Rendered as a section element\n</Box>`}>
  <BoxAsDemo client:load />
</Example>
