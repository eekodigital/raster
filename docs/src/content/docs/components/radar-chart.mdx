title: RadarChart
description: Multi-axis comparison chart. Natural fit for WCAG POUR principles (Perceivable, Operable, Understandable, Robust) or any multi-dimensional scoring. Keyboard navigable with screen reader fallback.
---

## Basic

  code={`import { RadarChart } from '@eekodigital/raster';\n\n<RadarChart\n  axes={['Perceivable', 'Operable', 'Understandable', 'Robust']}\n  series={[{ name: 'Current', data: [80, 65, 90, 70] }]}\n  aria-label="POUR principle scores"\n/>`}

## Multi-series comparison

Compare current state against a target or another audit.

<Example
  code={`<RadarChart\n  axes={['Perceivable', 'Operable', 'Understandable', 'Robust']}\n  series={[\n    { name: 'Current', data: [80, 65, 90, 70], color: 'var(--color-interactive)' },\n    { name: 'Target', data: [100, 100, 100, 100], color: 'var(--color-success)' },\n  ]}\n  max={100}\n  aria-label="Current vs target"\n/>`}
>
  <RadarChartMultiDemo client:load />
</Example>
