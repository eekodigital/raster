## In context

Badges typically appear alongside a label to communicate counts or states at a glance.

## When to use

<div class="callout">
  <p>
    <strong>Use Badge for</strong> compact numeric counts and single-word statuses that sit inline
    alongside a label, nav item, or table cell — unread messages, open findings, pass/fail
    indicators.
  </p>
  <p>
    <strong>
      Use <a href="/components/tags">Tag</a> instead
    </strong>{" "}
    for longer text labels or multi-word statuses where a pill with more breathing room is
    appropriate.
  </p>
</div>
