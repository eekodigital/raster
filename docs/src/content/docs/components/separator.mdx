## Horizontal

The default orientation. Spans the full width of its container.

<Example code={`import { Separator } from '@eekodigital/raster';\n\n<Separator />`}>
  <SeparatorHorizontalDemo client:load />
</Example>

## Vertical

Set `orientation="vertical"` to divide inline content. The separator fills the height of its flex container.

<Example code={`<Separator orientation="vertical" />`}>
  <SeparatorVerticalDemo client:load />
</Example>

## Decorative vs semantic

By default `decorative={true}` renders the element with `role="none"` — screen readers skip it.
Set `decorative={false}` when the divider conveys meaningful structure; it will then carry `role="separator"`.
